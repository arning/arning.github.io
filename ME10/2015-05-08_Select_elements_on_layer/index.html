Select elemets on layers in ME10/Creo Elements/Direct drafting

If it is required to select elements (TEXTS,DIMENSIONS, etc...) on a specific layer in ME10, it is possible to use the macro script language to select those elements.

Examples

  1. Select all texts.
    SELECT GLOBAL TEXTS NAMESPACE_INFOS 'SYS:LAYER_SCOPE' 'LAYER: [Layer name]' CONFIRM

  2. Select all dimensions.
    SELECT GLOBAL DIMENSIONS NAMESPACE_INFOS 'SYS:LAYER_SCOPE' 'LAYER: [Layer name]' CONFIRM

  3. Select all leader lines.
    SELECT GLOBAL LEADER_LINES NAMESPACE_INFOS 'SYS:LAYER_SCOPE' 'LAYER: [Layer name]' CONFIRM
